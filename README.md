# FinEase Client

FinEase Client is the frontend of a personal finance manager app. It helps users **track income and expenses**, **generate reports**, and **manage financial goals** through a simple, minimal, and secure interface.



## Features
- **Track Transactions:** Quickly record income and expenses with categories, description, and date.
- **Insightful Reports:** Generate reports by category and period to analyze spending.
- **Secure & Sync:** Firebase Auth ensures your data is private and synced across devices.
- **Minimal Interface:** Add transactions in seconds with smart defaults.

## Technologies Used
- React
- Vite
- Tailwind CSS (DaisyUI)
- Firebase Auth
