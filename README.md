## Features
- Track Transactions: Record income and expenses with categories, description, and date.
- Insightful Reports: Generate reports by category and period to analyze spending habits.
- Secure & Sync: Firebase Auth keeps your data private and synced across devices.
- Minimal Interface: Add transactions quickly with smart defaults.
